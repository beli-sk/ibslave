iB Slave
========

1-wire slave implementation for AVR emulating serial ID iButton (1990A/R).

Written for gcc-avr and avr-libc.

Development status: **Pre-Alpha**

[Project page](https://github.com/beli-sk/ibslave) is hosted on Github.
